Statistics Assignment

PROBLEM:

Using Ad Performance Report (attached), do A/B Testing of Ads within each ad group
For each ad group, find the ads that are losers, and should be removed. Use no of Impressions as the count of trials, and no of clicks as count of success.
Use 95% statistical confidence to determine if a given ad is better than another ad in the same ad group. Assume normal distribution.
Example of A/B Testing:
if there are 5 ads in an ad group, if 2 of them `win` over the all the other 3, the `other 3` are losers, and should be suggested for removal. Of course there can be ad groups where there are no clear winners or losers, and these should be marked.
OUTPUT:
A TSV with AdGroupID, and Ad Id of all the ads that are clear winners or losers. Winning Ads should be marked ‘WINNER’ and losing Ads should be marked ‘LOSER’. Ads that are neither winning or losing should be marked “NO_RESULT”
INPUT: A Ad Performance Report. It has AdGroupId, AdId, Clicks, and Impressions. (attached with email). Note that if you open the file in excel, it might show the IDs incorrectly in scientific notation.

Approach:

We have used Two Proportion Z Test with 95 % Confidence interval method.
For Z test to perform, as per stastics world, there are few sufficiency conditions to check whether the data is large enough to perform the Z test. The options for those sufficiency conditions are provded in the statistics page section where user can select the option, and results will be calculated accordingly.
